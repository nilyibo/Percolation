Epidemic Graph

About

Epidemic Graph is an interactive graph modeling the spread of epidemics.

App

Variables for selected vectex:

Infected?
Threshold

Directions:

  • Click in the open space to add a state
  • Drag between states to add a transition
  • Ctrl-drag a state to move graph layout
  • Click a state or a transition to select it
  • When a state is selected, delete removes the state
